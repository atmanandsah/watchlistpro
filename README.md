# Watchlist Pro for Upstox

Advanced, persistent, and lightning-fast watchlists built specifically for the Upstox Trading Terminal (`tv.upstox.com`).

## 🚀 Features

- ✅ **Unlimited Watchlists**: Create as many custom watchlists as you want. No artificial limits.
- ⚡ **Instant Chart Switching**: Uses Upstox's native TradingView API to switch charts instantly without ever reloading the page.
- 🖱️ **Native Sidebar Integration**: Injects seamlessly into the Upstox layout, pushing the chart cleanly to the side so your view is never obstructed.
- ⌨️ **Rapid Keyboard Navigation**: Use the `Up` and `Down` arrow keys to rapidly flip through your entire watchlist.
- ✋ **Drag & Drop Reordering**: Grab any stock and drag it to customize your exact list order.
- ☁️ **Unlimited Cloud Sync**: Automatically backs up all your watchlists to your Google Account or Brave Sync Chain (Utilizes an advanced chunking algorithm to completely bypass Chrome's 8KB sync limits).
- 🔒 **100% Secure & Private**: The extension does not track you, does not connect to external databases, and has absolutely no access to your Upstox account credentials or funds. All data stays natively in your browser storage.

## 🛠️ Installation (Developer Mode)

1. Download or clone this repository to your computer.
2. Open your Chromium browser (Chrome, Brave, Edge) and go to `chrome://extensions`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** in the top left corner.
5. Select the `extension` folder from this repository.
6. Open `tv.upstox.com` and click the extension icon to toggle the sidebar!
